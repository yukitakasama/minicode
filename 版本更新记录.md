# Minicode 版本更新记录

## 2026-07-22

### 工作界面快捷键

新增常用工作界面快捷键：

| 快捷键 | 功能 |
|---|---|
| `Ctrl/Cmd + B` | 展开或折叠侧栏 |
| `Ctrl/Cmd + L` | 聚焦聊天输入框 |
| `Ctrl/Cmd + W` | 关闭当前 Tab，并复用运行中会话确认流程 |
| `Ctrl/Cmd + Tab` | 切换到下一个 Tab |
| `Ctrl/Cmd + Shift + Tab` | 切换到上一个 Tab |
| `Ctrl/Cmd + 1~9` | 跳转到对应序号的 Tab |
| `Ctrl/Cmd + J` | 打开或关闭当前会话 Terminal 面板 |
| `Ctrl/Cmd + Shift + E` | 打开或关闭当前会话 Workspace 面板 |
| `Ctrl/Cmd + Shift + A` | 打开或关闭当前会话 Activity 面板 |
| `Ctrl/Cmd + ,` | 打开设置 |

### 保留的既有快捷键

- `Ctrl/Cmd + N`：新建会话
- `Ctrl/Cmd + K`：全局会话搜索
- `Ctrl/Cmd + F`：页面内搜索
- `Ctrl/Cmd + .`：停止当前生成
- `Ctrl/Cmd + +/-/0`：调整或重置应用缩放
- `Enter` / `Shift + Enter`：发送消息或插入换行

### 实现说明

- 快捷键统一通过 `minicode:keyboard-shortcut` 事件分发。
- Tab 切换和关闭逻辑复用现有 `tabStore` 与 TabBar 的关闭确认机制。
- 文本输入框、可编辑区域和 Select 控件中不会拦截工作区级快捷键。
- 未修改用户配置文件和既有快捷键行为。

### 权限与发送快捷键

- `Enter`：在权限确认中执行“允许”；在聊天输入框中按当前发送设置发送消息。
- `Alt+Enter`：在通用工具权限确认中执行“本次会话允许”。
- 发送按钮、允许按钮和计算机操作权限按钮的悬停提示会显示对应快捷键。
- 设置页消息发送区域增加工作界面快捷键集中说明。
