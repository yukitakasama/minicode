## 2026-07-23

### 修复 Windows 聊天框无法粘贴文件上传（#1086）

- 根因：composer 粘贴逻辑只处理剪贴板图片，忽略资源管理器复制的普通文件（如 .md）
- 修复：统一从 clipboardData.files / kind:file 收集文件，走既有 filesToComposerAttachments 路径附件链路
- ChatInput 与 EmptySession 同步支持；截图粘贴与纯文本粘贴行为保持不变
- 影响范围：桌面端会话输入框、首页空白会话输入框
- 验证：composerAttachments / ChatInput / EmptySession 相关 Vitest 通过

## 2026-07-22

### 修复大附件导致 Renderer 白屏崩溃（#1087）

- 桌面端通过 Electron webUtils.getPathForFile 解析本地文件路径，避免整文件 base64 进 Renderer

- 桌面非图片附件禁止在无 path 时内联；任意内联附件硬上限 12MB

- 发送/UI 消息在已有 path 时剥离 data，防止 WebSocket/消息列表驻留超大字符串

- 影响范围：composer 附件、拖放/文件选择、消息发送载荷

- 验证：composerAttachments / ChatInput / desktopHost contract 测试通过

## 2026-07-22

### 修复非 Anthropic 格式接口 502 / 鉴权失败

- 根因：激活的 OpenAI 兼容服务商会把 `ANTHROPIC_BASE_URL` 设为 `http://127.0.0.1:<port>/proxy`，而本地访问令牌注入只匹配以 `/proxy/` 开头的路径，导致 CLI 请求未带桌面本地令牌，代理能力路由鉴权失败。

- 修复：本地代理路径识别兼容 `/proxy` 与 `/proxy/providers/:id`；激活服务商托管 env 改为 provider 作用域路径。

- 影响范围：`openai_chat` / `openai_responses` 等非 Anthropic 格式接口；Anthropic 直连不受影响。

- 验证：`client.test.ts`、`provider-runtime-env.test.ts`、`proxy-upstream-url.test.ts` 通过。

### 修复对话标签关闭叉号不可见

- 根因：关闭按钮使用 `opacity-0` + `group-hover`，在桌面拖拽区/悬停态下图标不可见，但命中区域仍可点击。

- 修复：关闭按钮始终可见，图标改为 Lucide `X`。

- 影响范围：顶部对话标签栏。

- 验证：`TabBar.test.tsx` 38 项通过。

## 2026-07-22

### 工作界面快捷键

新增常用工作界面快捷键：

| 快捷键 | 功能 |

|---|---|

| `Ctrl/Cmd + B` | 展开或折叠侧栏 |

| `Ctrl/Cmd + L` | 聚焦聊天输入框 |

| `Ctrl/Cmd + W` | 关闭当前 Tab，并复用运行中会话确认流程 |

| `Ctrl/Cmd + Tab` | 切换到下一个 Tab |

| `Ctrl/Cmd + Shift + Tab` | 切换到上一个 Tab |

| `Ctrl/Cmd + 1~9` | 跳转到对应序号的 Tab |

| `Ctrl/Cmd + J` | 打开或关闭当前会话 Terminal 面板 |

| `Ctrl/Cmd + Shift + E` | 打开或关闭当前会话 Workspace 面板 |

| `Ctrl/Cmd + Shift + A` | 打开或关闭当前会话 Activity 面板 |

| `Ctrl/Cmd + ,` | 打开设置 |

### 保留的既有快捷键

- `Ctrl/Cmd + N`：新建会话

- `Ctrl/Cmd + K`：全局会话搜索

- `Ctrl/Cmd + F`：页面内搜索

- `Ctrl/Cmd + .`：停止当前生成

- `Ctrl/Cmd + +/-/0`：调整或重置应用缩放

- `Enter` / `Shift + Enter`：发送消息或插入换行

### 实现说明

- 快捷键统一通过 `minicode:keyboard-shortcut` 事件分发。

- Tab 切换和关闭逻辑复用现有 `tabStore` 与 TabBar 的关闭确认机制。

- 文本输入框、可编辑区域和 Select 控件中不会拦截工作区级快捷键。

- 未修改用户配置文件和既有快捷键行为。

### 权限与发送快捷键

- `Enter`：在权限确认中执行“允许”；在聊天输入框中按当前发送设置发送消息。

- `Alt+Enter`：在通用工具权限确认中执行“本次会话允许”。

- 发送按钮、允许按钮和计算机操作权限按钮的悬停提示会显示对应快捷键。

- 设置页消息发送区域增加工作界面快捷键集中说明。
